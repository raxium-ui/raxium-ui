# v-title

`v-title` 是 addons 提供的文本提示指令，用于在鼠标悬停时展示浮层标题内容，并支持尺寸与表面主题切换。该能力属于 addons 独立实现（非 Ark UI 对齐组件），不提供组件级 `theme` / `ui` 扩展入口；可通过指令参数与样式覆盖进行扩展。

## Theme Configuration

`v-title` 不提供组件级 `theme` 配置入口。可通过指令参数（如 <code>v-title:dark</code>）切换内置表面属性，并配合样式覆盖扩展更多主题表现。

---

## UI Configuration

`v-title` 不提供 `ui` 或 `.ui.*` 配置能力。结构与视觉定制建议通过宿主元素 class 与 <code>.rui-v-title</code> 样式覆盖完成。

## 最佳实践

- 为避免无意义提示，建议仅在文本可能被截断或信息需要补充说明时使用 `v-title`。
- 需要更强对比度时优先使用 `v-title:dark`，并与页面背景对照验证可读性。
- 列表或高频悬停场景下，保持 tooltip 内容为短文本，减少频繁重排与定位抖动。
- 若需统一大字号提示，优先使用 `.lg` 而不是业务侧重复定义字体样式，保持行为与样式一致。
- 对非内置主题值（如 `v-title:brand`），请同步补充 <code>[data-theme-surface='brand']</code> 样式规则，避免只有属性变化而无视觉反馈。
- 对不需要悬浮样式的节点，不建议使用无效 modifier（如当前实现未消费的 <code>.unstyled</code>），避免产生“已配置但无效果”的认知偏差。
