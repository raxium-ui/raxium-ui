# Slider

`Slider` 是用于区间数值选择的滑动输入组件，支持单值与多 Thumb（范围）模式、水平/垂直方向、刻度标记与 Tooltip 展示。组件基于 Ark UI Slider 构建，根组件在透传 Ark UI Root 常用能力的同时，叠加 Raxium UI 的 **`theme` / `craft`**、`ui` 与各子组件上的 **`theme`（仅 variant，无 `craft`）**。

## ARK UI Link

`Slider` 组件族与 Ark UI Slider 的关系如下：

- `Slider` 根组件基于 `Slider.RootProvider + useSlider`，根 props 与事件透传 Ark UI Root 能力（如 `modelValue/defaultValue`、`min/max/step`、`orientation`、`origin`、`minStepsBetweenThumbs` 等）。
- `SliderThumb`、`SliderMarker` 基于 Ark UI 对应原语封装，保留原语属性并补充主题样式。
- `SliderTooltipThumb`、`SliderTooltipMarker` 通过组合 Ark Slider 与 Tooltip 提供“值提示”能力，属于 Raxium UI 扩展，不是 Ark UI 一一对应子组件。

- **相关文档**: [Ark UI Slider (Vue)](https://ark-ui.com/vue/docs/components/slider)

## Theme Configuration

`Slider` 根由内层 **`ThemeProvider`** 把合并后的 **`theme`** 上下文提供给 Thumb、Marker、Tooltip 系列子组件；全局 **`tvSlider`** 合并顺序与 `useTheme` 一致（默认值 ← `RUIConfigProvider` / `ThemeProvider` ← 根 `theme`，再由根 **`craft`** 覆写该组件实例的 slider craft）。

- **`theme`（根与子组件共用语义）**：仅 **`skin`、`surface`、`size`、`unstyled`、`bordered`** 等与 `ThemeProps` 对齐的字段，**不要把 `crafts` 写在组件的 `theme` prop 上**；应用级批量改 craft 请用 **`RUIConfigProvider` / preset 的 `theme.crafts`**。
- **`craft`（仅根 `Slider`）**：`CraftOverride<'tvSlider'>`，用于调整 `tvSlider` 的槽位与变体（见下表槽位列表）。
- **`ThemeNoCrafts` 子组件**（`SliderThumb`、`SliderMarker`、`SliderTooltipThumb`、`SliderTooltipMarker`）：只可传 **variant 形式的 `theme`**，**没有根级 `craft` prop**；样式结构仍来自同一套 **`tvSlider`**。

`tvSlider` 槽位：<code>root</code> / <code>control</code> / <code>track</code> / <code>range</code> / <code>thumb</code> / <code>marker</code> / <code>markerDot</code> / <code>markerValue</code>。根上通过 `useCraft(..., { orientation })` 会把 Ark 的 **`orientation`** 一并喂给变体（水平/垂直布局）。

## 最佳实践

- 单值与范围统一使用数组值（如 `[30]`、`[20, 70]`），避免在业务层混用标量与数组造成状态分支复杂化。
- 需要联动外部筛选状态时使用受控模式（`v-model` + `update:modelValue`）；仅做初始展示时优先 `defaultValue`。
- 范围模式务必为每个 Thumb 明确设置 `index`，并在需要时配置 `minStepsBetweenThumbs` 防止手柄过度重叠。
- 垂直布局不要只改 `orientation`，同时调整容器高度与 `ui.control/ui.track/ui.range`，保证命中区与视觉一致。
- Tooltip 建议用于精确反馈场景：高频拖拽时可监听 `valueChange` 做实时预览，提交动作优先放在 `valueChangeEnd`。
- 样式定制分层：单次布局/间距优先 **`ui.*` / `class`**；整条产品线改尺寸与槽位语义优先 **`theme.size`**（及 Provider）；需要改 **`tvSlider`** 结构时再使用根 **`craft`** 或全局 **`theme.crafts`** / preset。
